# MediaMatrix — Digital Asset Management

A Shutterstock-like platform for uploading, organizing, and discovering digital assets.

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Search: ElasticSearch
- Storage: Firebase + CloudFront CDN
